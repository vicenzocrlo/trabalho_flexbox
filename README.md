# Atividade Avaliativa Flexbox
## O objetivo dessa atividade foi a criação de um simples site usando as seguintes diretrizes:
1. No head do html: utf-8, link com css, link com ícone, descrição, palavra-chave, autor

2. No body: estruturado com .container, header, nav, main, section, aside, footer (vc poderá criar mais caixas como fizemos com .conteúdo)

3. Formate toda a estrutura com CSS, utilize cores de fundo ou não, cores de fonte, margens, padding, flexbox (display:flex)

4. No header colocar uma imagem e o nome da empresa (um do lado do outro)

5. No main coloque section e aside lado a lado

6. No section insira texto de parágrafo e imagem
